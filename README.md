A smart, AI-powered meal planner that turns your pantry or vegetable box deliveries into recipes and a shopping list.

Try it out on github pages: [AI Recipe Planner](https://drdonik.github.io/ai-recipe-planner/)

## Features

- **AI-Powered Recipes**: Generates personalized recipes using Google Gemini Flash 3 Preview.
- **Minimising food wste**: Input vegetables, ingredients, spices and staples you have to minimize food waste.
- **Customizable**: Set your dietary preferences (Vegan, Vegetarian, Pescatarian, etc.), style or other wishes, number of people, and number of meals.
- **Smart Shopping List**: Automatically generates a shopping list for missing ingredients.
- **Multi-language**: Supports for multiple languages.
- **Responsive Design**: Glassmorphism UI that works on desktop and mobile. Fully supported dark mode for comfortable viewing at night.

## Getting Started

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```

## Usage

1.  Enter your API Key (Gemini) in the header.
2.  Add ingredients you have in your pantry.
3.  Select your diet, number of people, and number of meals.
4.  Click **Generate** to get your personalized menu!
